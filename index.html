기본적인 매크로

  • {{pipe}}
    슬래시 명령어 배치를 위한 것. 이전 명령어의 결과로 대체됨.
  • {{newline}}
    줄 바꿈 추가.
  • {{trim}}
    주변 줄 바꿈 제거.
  • {{noop}}
    아무 작업도 하지 않음. 빈 문자열 반환.
  • {{original}}
    API 설정에서 정의된 글로벌 프롬프트. (고급 정의 프롬프트에서만 유효)
  • {{input}}
    사용자 입력.
  • {{lastGenerationType}}
    마지막 생성 요청의 유형. (없으면 빈 값) 가능한 값: "normal", "impersonate", "regenerate", "quiet", "swipe", "continue".
  • {{charPrompt}}
    캐릭터의 메인 프롬프트 오버라이드.
  • {{charInstruction}}
    캐릭터의 포스트-히스토리 지침.
  • {{description}}
    캐릭터 설명.
  • {{personality}}
    캐릭터 성격.
  • {{scenario}}
    캐릭터의 시나리오.
  • {{persona}}
    현재 사용자의 페르소나 설명.
  • {{mesExamples}}
    캐릭터의 대화 예제.
  • {{mesExamplesRaw}}
    포맷되지 않은 대화 예제 (스토리 스트링 전용).
  • {{summary}}
    "Summarize" 확장에서 생성된 최신 채팅 요약.
  • {{user}}
    현재 사용자의 이름.
  • {{char}}
    캐릭터 이름.
  • {{char_version}}
    캐릭터의 버전 번호.
  • {{group}}
    그룹 채팅의 모든 멤버 이름을 쉼표로 구분하여 나열. (솔로 채팅 시 캐릭터 이름 표시)
  • {{groupNotMuted}}
    그룹 내에서 음소거된 멤버 제외한 목록.
  • {{model}}
    현재 선택된 API의 텍스트 생성 모델 이름. (정확하지 않을 수 있음)
  • {{lastMessage}}
    가장 최근의 채팅 메시지.
  • {{lastUserMessage}}
    가장 최근의 사용자 메시지.
  • {{lastCharMessage}}
    가장 최근의 캐릭터 메시지.
  • {{lastMessageId}}
    최근 채팅 메시지의 인덱스 번호.
  • {{firstIncludedMessageId}}
    현재 컨텍스트에 포함된 첫 번째 메시지의 ID.
  • {{currentSwipeId}}
    마지막 메시지의 현재 스와이프 ID. (사용자 메시지나 숨겨진 메시지일 경우 빈 값)
  • {{lastSwipeId}}
    마지막 메시지의 스와이프 개수. (사용자 메시지나 숨겨진 메시지일 경우 빈 값)
  • {{reverse:(내용)}}
    해당 내용을 반대로 변환.
  • {{// (메모)}}
    AI에게 보이지 않는 메모 추가. 빈 값으로 대체됨.
  • {{time}}
    현재 시간.
  • {{date}}
    현재 날짜.
  • {{weekday}}
    현재 요일.
  • {{isotime}}
    현재 ISO 형식의 시간 (24시간제).
  • {{isodate}}
    현재 ISO 형식의 날짜 (YYYY-MM-DD).
  • {{datetimeformat ...}}
    특정 형식으로 날짜/시간 표시 (예: {{datetimeformat DD.MM.YYYY HH:mm}} → 독일식 날짜/시간 형식).
  • {{time_UTC±#}}
    특정 UTC 시간대 기준 현재 시간 (예: UTC-4 또는 UTC+2).
  • {{timeDiff::(시간1)::(시간2)}}
    두 시간 간의 차이 계산 (예: {{timeDiff::{{isodate}} {{time}}::2024/5/11 12:30:00}}).
  • {{idle_duration}}
    마지막 사용자 메시지 이후 경과한 시간.

AI 생성 조정 매크로

  • {{bias "텍스트"}}
    다음 사용자 입력 전까지 AI의 행동 편향 설정.
  • {{roll:(주사위 공식)}}
    주사위를 굴림. (예: {{roll:1d6}} → 1~6 중 랜덤 값 반환)
  • {{random:(항목들)}}
    주어진 목록에서 랜덤 항목 선택 (예: {{random:1,2,3,4}} → 1~4 중 하나 선택).
  • {{random::(항목1)::(항목2)}}
    쉼표를 포함한 항목도 랜덤 선택 가능.
  • {{pick::(항목들)}}
    {{random}}과 유사하지만, 한 번 선택되면 해당 채팅에서 계속 유지됨.
  • {{banned "텍스트"}}
    특정 단어를 금지어 목록에 동적으로 추가 (Text Generation WebUI에서만 사용 가능).

환경 감지 매크로

  • {{isMobile}}
    모바일 환경에서는 "true", 아니면 "false" 반환.

고급 모드 및 컨텍스트 템플릿 매크로

(고급 서식 설정에서 활성화해야 함)

  • {{maxPrompt}}
    허용된 최대 프롬프트 길이 (토큰 단위).
  • {{exampleSeparator}}
    컨텍스트 템플릿 대화 예제 구분자.
  • {{chatStart}}
    컨텍스트 템플릿의 채팅 시작 지점.
  • {{systemPrompt}}
    시스템 프롬프트 내용.
  • {{defaultSystemPrompt}}
    기본 시스템 프롬프트 내용.
  • {{instructSystemPromptPrefix}}
    시스템 프롬프트 접두어.
  • {{instructSystemPromptSuffix}}
    시스템 프롬프트 접미어.
  • {{instructUserPrefix}}
    사용자 입력 접두어.
  • {{instructUserSuffix}}
    사용자 입력 접미어.
  • {{instructAssistantPrefix}}
    AI 응답 접두어.
  • {{instructAssistantSuffix}}
    AI 응답 접미어.
  • {{instructFirstAssistantPrefix}}
    첫 번째 AI 응답 접두어.
  • {{instructLastAssistantPrefix}}
    마지막 AI 응답 접두어.
  • {{instructSystemPrefix}}
    시스템 메시지 접두어.
  • {{instructSystemSuffix}}
    시스템 메시지 접미어.
  • {{instructSystemInstructionPrefix}}
    시스템 명령 접두어.
  • {{instructUserFiller}}
    첫 번째 사용자 입력 필러.
  • {{instructStop}}
    AI가 응답을 멈추는 구간.
  • {{instructFirstUserPrefix}}
    첫 번째 사용자 입력 접두어.
  • {{instructLastUserPrefix}}
    마지막 사용자 입력 접두어.

채팅 변수 매크로

로컬 변수 (현재 채팅 내에서만 유지)

  • {{getvar::이름}}
    해당 로컬 변수의 값 반환.
  • {{setvar::이름::값}}
    해당 로컬 변수에 값 설정 (출력 없음).
  • {{addvar::이름::증가량}}
    해당 변수의 값을 증가.
  • {{incvar::이름}}
    해당 변수 값을 1 증가.
  • {{decvar::이름}}
    해당 변수 값을 1 감소.

글로벌 변수 (모든 채팅에서 유지)

  • {{getglobalvar::이름}}
    해당 글로벌 변수의 값 반환.
  • {{setglobalvar::이름::값}}
    해당 글로벌 변수에 값 설정.
  • {{addglobalvar::이름::값}}
    해당 글로벌 변수의 값을 증가.
  • {{incglobalvar::이름}}
    해당 글로벌 변수 값을 1 증가.
  • {{decglobalvar::이름}}
    해당 글로벌 변수 값을 1 감소.

스코프 변수 (STscript 내에서 유지)

  • {{var::이름}}
    해당 스코프 변수의 값 반환.
  • {{var::이름::인덱스}}
    리스트나 딕셔너리에서 특정 인덱스 또는 키에 해당하는 값 반환.
