예시: /audioimport type=bgm https://ex.com/song1.mp3, https://ex.com/song2.mp3 BGM 음악을 가져와서 첫 번째 링크를 즉시 재생합니다. /audioimport type=ambient play=false url=https://ex.com/sound1.mp3, https://ex.com/sound2.mp3 효과음 링크를 가져옵니다 (자동 재생 안 함).

/if
왼쪽 피연산자 a와 오른쪽 피연산자 b의 값을 비교하고, 조건이 참이면 따옴표로 묶인 유효한 슬래시 명령을 실행하고 명령 실행 결과를 파이프로 전달합니다. 왼쪽과 오른쪽 피연산자에 숫자 값과 문자열 리터럴이 지원됩니다. 규칙이 제공되지 않으면 기본값은 eq입니다. 오른쪽 피연산자가 제공되지 않으면 왼쪽 값이 참인지 확인하는 것으로 기본 설정됩니다. 비어 있지 않은 문자열이나 0이 아닌 숫자, true나 on 값은 참으로 간주됩니다. 오른쪽 피연산자가 제공되지 않은 경우 허용되는 규칙은 not과 규칙 없음(기본값으로 참인지 여부 반환)뿐입니다. 사용 가능한 규칙: eq => a == b (문자열 및 숫자) neq => a !== b (문자열 및 숫자) in => a에 b 포함 (문자열로서의 문자열 및 숫자) nin => a에 b가 포함되지 않음 (문자열로서의 문자열 및 숫자) gt => a > b (숫자) gte => a >= b (숫자) lt => a < b (숫자) lte => a <= b (숫자) not => !a (참/거짓)

/while
왼쪽 피연산자 a와 오른쪽 피연산자 b의 값을 비교하고, 조건이 참이면 따옴표로 묶인 유효한 슬래시 명령을 실행합니다. 왼쪽과 오른쪽 피연산자에 숫자 값과 문자열 리터럴이 지원됩니다.

사용 가능한 규칙:
eq => a == b (문자열 및 숫자)
neq => a !== b (문자열 및 숫자)
in => a에 b 포함 (문자열로서의 문자열 및 숫자)
nin => a에 b가 포함되지 않음 (문자열로서의 문자열 및 숫자)
gt => a > b (숫자)
gte => a >= b (숫자)
lt => a < b (숫자)
lte => a <= b (숫자)
not => !a (참/거짓)

예시: /setvar key=i 0 | /while left=i right=10 rule=lte "/addvar key=i 1" "i" 값이 10에 도달할 때까지 "i" 값에 1을 추가합니다. /while left={{getvar::currentword}} {: /setvar key=currentword {: /do-something-and-return :}() | /echo 현재 단어는 "{{getvar::currentword}}"입니다 :} "currentword" 변수가 참(거짓/비어 있지 않은 내용)인 한 정의된 하위 명령을 실행합니다. 루프는 기본적으로 100회 반복으로 제한됩니다. guard=off를 전달하여 비활성화할 수 있습니다.

리턴값: 마지막으로 실행된 명령의 결과
